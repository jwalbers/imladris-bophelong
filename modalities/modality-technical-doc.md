# DICOM Modality Technical Documentation Reference

**Purpose:** Online technical documentation resources for medical imaging modalities identified in the landscape assessment.  
**Last Updated:** May 2026  

> **Note:** Some of these modalities are legacy or military-fielded systems. Where direct manufacturer documentation portals exist, those are listed first. Where documentation is hosted on third-party or archive sites, those are noted accordingly. For proprietary service manuals, contacting the manufacturer's technical support is recommended.
